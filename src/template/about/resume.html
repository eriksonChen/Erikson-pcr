Biography/自傳

陳柏達 / Erikson

第一次踏入網路產業是從網頁設計開始，對設計的美感有自己的看法，接著設計做了一兩年轉換跑道開始摸索程式的樂趣。

我是從flash盛行的年代開始接觸寫程式這一條路,那時候寫as3是網頁前端的主流，也是一個美好的年代，不用考慮所有瀏覽器的相容性跟所有的行動裝置,主要是負責網頁上的互動及流程的串接作製作，並與後端配合串接api存取資料庫的資料。

隨著flash式微，開始學習javascript及其他相關工具，除了前端網頁技術外，也熟悉FB api , Angular js, Git 版本控管及glup開發工具，上一個工作的專案大多用Angular JS 框架去完成，和fb api 串接也是幾乎每個案子都會用到。
